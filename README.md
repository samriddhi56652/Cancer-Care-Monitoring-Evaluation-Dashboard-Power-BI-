Project Overview

This project presents a comprehensive Monitoring & Evaluation (M&E) dashboard for analyzing cancer care delivery across districts. The dashboard evaluates healthcare performance using an Input → Process → Output framework, integrating People, Process, and Technology dimensions.

The objective is to identify system inefficiencies, resource gaps, and outcome drivers to support data-driven healthcare decision-making.

Key Objectives

Evaluate cancer care performance across districts

Analyze patient flow from screening to treatment

Identify bottlenecks in diagnosis and treatment

Assess impact of resources (oncologists, infrastructure) on outcomes

Provide actionable insights for improving survival rates

Key Insights

High treatment completion (83%) indicates strong adherence once treatment begins

Significant drop-offs occur before treatment, especially during diagnosis stages

Early detection strongly correlates with improved survival outcomes

Increased oncologist availability shows limited impact beyond a threshold, indicating inefficiencies

Mortality is concentrated in specific districts, suggesting regional disparities

Patient satisfaction is consistent but not exceptional, indicating scope for improvement

Key Challenges Identified

Delays in diagnosis and treatment initiation

Late-stage detection (higher Stage III–IV cases)

Uneven resource utilization across districts

Financial barriers (low insurance vs high treatment cost)

Recommendations

Strengthen early screening programs

Reduce diagnostic and treatment delays

Improve insurance coverage and affordability

Expand digital health systems and tracking

Optimize resource allocation based on performance gaps

Data Limitations

The dataset lacks key variables that could enhance analysis:

Patient demographics (age, gender, lifestyle risks)

Dropout reasons (financial, accessibility, awareness)

Time-stamped patient journey data

Staff workload and efficiency metrics

Data quality and reporting accuracy indicators
